Данный проект создан для парсинга вакансий и работников с сайта hh.ru
Чтобы пользоваться данным приложением надо:
1)Иметь скачанный Docker
2)Чтобы запустить проект введите команду docker-compose up --build
3)Чтобы остановить проект введите команду docker-compose down
4)Чтобы перезапустить проект docker-compose restart
